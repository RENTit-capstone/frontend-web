## About
RENTit의 어드민용 대시보드를 위한 프론트엔드 레포지토리입니다.


## 실행 방법

아래의 명령어를 입력하여 사용하는 라이브러리를 설치하고:
```
npm install
```

아래의 명령어로 실행한 후:
```
npm run dev
```

아래의 주소로 진입하세요:
`localhost:5173`
